# unswirl

### Requirements
* Opencv
* Opencv_contrib

### Build OpenCV
Make sure .env file is configured, then run
```
./prebuild.sh
```

### Build and run unswirl
Make sure .env file is configured, then run
```
./build.sh
```

Run examples
```
target/debug/unswirl examples/src1.png examples/dst1.png
target/debug/unswirl examples/src2.png examples/dst2.png
```
