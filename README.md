# unswirl

### Requirements
* Opencv
* Opencv_contrib

### Build OpenCV
Make sure .env file is configured, then run
```
./prebuild.sh
```

### Build and run unswirl
Make sure .env file is configured, then run
```
./build.sh
```

Run examples
```
target/debug/vizion unskew examples/src1.png examples/dst.png
```
